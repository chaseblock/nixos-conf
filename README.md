# Chase's Nix Flake

Based on [Theo's Nix Flake](https://github.com/theopn/nix-conf) (and others)

Still very much a work in progress.

## Notes

 - The image placed at ~/.wallpaper is used for the wallpaper and lockscreen.
 - The color scheme of many things is set using `wal`. Make sure to run this program
   after installation.
